## Introduction

一款现代化的后台管理脚手架项目，使用 Spring Boot 3 + Spring Security + JWT + Mybatis Plus + Redis 实现

## Features

- [x] 用户管理
- [x] 角色管理
- [x] 菜单管理
- [x] 权限管理
- [ ] 字典管理
- [ ] 事件日志
- [ ] 对象存储

## Requirements

- Eclipse Temurin 21
- MySQL 5.8 / PostgreSQL 17
- Maven 3.6.0+
